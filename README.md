# Portfolio Milan LOÏ

Portfolio professionnel réécrit en **React 19** & **Vite 8** avec une architecture cinématique avancée.

## ✨ Fonctionnalités clés
- **Constellation Antigravity** : Système de particules réactives développé sur mesure avec Canvas 2D natif, physique orbitale et évitement magnétique au survol du curseur.
- **Défilement fluide** : Intégration du moteur Lenis pour un défilement fluide d'inertie.
- **Rendu Responsive** : Adaptabilité complète aux formats mobiles et tablettes.
- **Simplification** : Base de code commentée de façon homogène en français.

## 🚀 Déploiement
Le projet utilise **GitHub Actions** pour le déploiement continu. À chaque push sur `main`, le code est compilé pour la production et publié en direct sur GitHub Pages.
